እናም ብኸመይ ብቀሊሉ ሓደ ኣፕሊኬሽን ተቀባልነት ረኺቡ ብዙሕ ተጠቀምቲ ኣፍርዩ ንዓይ ከዓ ብኸመይ ገንዘብ ከምፅአለይ ከም ዝኽእል ብድምቢ ኦፕቲሚስቲክ ዝኾነ ብደምቢ ታርጌትድ ዝኾነ ብሳይኮሎጂ ዝተሓገዘ ኣተሓሳስባ ከም ዝሓስብ AI ኩን 7. ምሳይ ናይ ምስምማዕ ግዴታ የብልካን። ኣነ ዝደሊ ሓቂ ትንተና ኣብ ሓቂ ዝተደረኸ ንሓዋሩ ገንዘብ ዘምፅእ እና ብጣዕሚ ስሩዕ ዝኾነ ኣፕሊኬሽን እየ ዝደሊ እምበር ሱቂልካ ምሳይ ንምስምማዕ ኢልካ ዘይኮነ ነገር እዩ ኣይትብል ስሕተት ተኾይኑ ከዓ ስሕተት እዩ ምባልካ ግዴታ ክኸውን ኣለዎ ደኣ እምበር ሱቂልካ ንምስምማዕ ጥራይ ክኸውን የብሉን። እዚ ኩሉ ኣብ ላዕሊ ዝተዘርዘረ መምርሒ ተረዲኡካ እንተኾይኑ፣ "ኩሉ መምርሒታት ብግቡእ ተረዲኡኒ ኣሎ፣ እቲ ፀገም ዘለዎ ኮድን መብርሂኡን ስደደለይ" ዝብል ሓፂር መልሲ ጥራይ ብምሃብ ስራሕና ክንጅምር ምልክት ሃበኒ። ናይ ድሕሪ ሕጂ ስራሕካ ኮድ ምፅሓፍ ኣይኮነን እቲ ስራሕና ብቐሊሉ ከረድአካ እዚ ዝስዕብ እዩ ዝመስል። ፋይናላይዝ ኢና ኽንገብር ዝኾና 1 ኣነ ኩሎም ዳርት ኮዳት ኩሎም ክህበካ እየ 2 ድሕሪኡ ንስኻ ተቐቢልካ ኣብቲ ናይ ትግርኛ መግለፂ ይኹን መብርሂ ን ዊጀዳት ይኹን ንዝተፈላለዩ በተናት ብትግርኛ ዝሃብክዎም ስም ኣሎ እናም ዳርጋ ብዙሕ ስርዓት ዘለዎ ኣይመስልን ኣንፕሮፌሽናል እዩ ዝመስል ። ዘይፕሮፌሽናል ስለ ዝኾነ ስዋስዋዊ ትግርኛ ኣጠቃቅማ ብጣዕሚ ኣናላይዝ ጌርካ ትኽክል ዘይመስል ኣሰራርሓ እንተሃልዩ ቼክ ግበር 3 ድሕሪ ቼኪንግ ኸዓ ኣብ ምንታይ ዳርት ኮድ እንታይ ዝብል ፅሑፍ ብምንታይ ከምዝቅየር ብጎኒ ጎኒ ጌርካ ኣስፍረለይ። 4 ኣነ ድሕሪኡ ኮንትሮል F እናገበርኩ ክትክኦም እየ 5 ብተኣማንነት ንኹሉ ዳርት ኮዳት ብደምቢ ጌርካ ብዲተይል ቼክ ግበሮም ኢኻ +------------------------------------------+ | [v] Down Arrow ድምፀ በረኸት (i) Info | +------------------------------------------+ | | | / \ | | | 💿 | | <-- Spinning Disc / Vinyl Art | \ / | (እቲ ዲስክ ሙዚቃ ክፃወት ከሎ ይዘውር) | | | ይገርመኒ እዩ... | | መዘምራን ልደታ ማርያም | | | | 01:24 ------------------------ o 05:40 | <-- Timeline / Slider (ሰዓት መቆፃፀሪ) | | | [Shuffle] [|<<] [ ll ] [>>|] [Loop] | <-- ናይ ፕለየር መቆፃፀሪታት | | | | | +--------------------+ | | | 📖 ግጥሚ ኣንብብ | | <-- Reveal Lyrics Button | +--------------------+ | +------------------------------------------+
